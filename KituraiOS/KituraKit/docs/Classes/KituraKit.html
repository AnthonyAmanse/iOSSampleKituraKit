KituraKit

public class KituraKit

A client side library for using REST requests in a web application. KituraKit allows developers to use Swift 4’s Codable protocol in their front and back end applications and use the same code on the front and backend. It also provides a Create, Read, Update and Delete (CRUD) API to make calls to servers easy and as swift-y as possible
